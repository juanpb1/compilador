# Compilador

Compilador criado para avaliação da disciplina de Compiladores. Abaixo estão as regras de como o compilador deve se comportar.

## Identificadores

Exemplos de alguns identificadores válidos:
- `i`
- `jx`
- `soma`
- `k12aa3b`
- `z_teste`
- `u_teste123ab`
- `u_12tes3`

Exemplos de alguns identificadores inválidos:
- `hhz_nota`
- `z_b_bb`
- `bb_`
- `2abc`

## Palavras Reservadas

- `principal`
- `if`
- `then`
- `else`
- `while`
- `do`
- `until`
- `repeat`
- `int`
- `double`
- `char`
- `case`
- `switch`
- `end`
- `procedure`
- `function`
- `for`
- `begin`
- `boolean`
- `type`
- `var`

- `}`

## Comentários

- `@@` -> Comentário de uma linha
- `// comentário de várias linhas //`
- `!!` -> Comentário de uma linha
- `! comentário de várias linhas !`

- `2`
- `345`
- `23.5`
- `5.55`
- `-4.5`
